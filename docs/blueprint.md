# **App Name**: CropCast

## Core Features:

- Data Input Form: Form-based input for weather data, soil properties, crop management, crop variety, and pest/disease info.
- Yield Prediction: ML model predicts crop yield (kg/hectare) based on input data using a pre-trained model. This model uses tool use for identifying important factors
- Data Visualization: Visualization of predicted yield (graph/chart) and input data (bar/pie charts).
- Yield Output Display: Displays the predicted crop yield result, formatted for clarity.
- Data Preprocessing: Uses data from uploaded and pre-existing datasets like FAO, USDA, or CGIAR to make an efficient decision. No external DB.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to reflect agriculture and growth.
- Background color: Light beige (#F5F5DC) to provide a neutral and clean backdrop.
- Accent color: Warm yellow (#FFD700) to highlight important data and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif font that's modern but approachable and suitable for both headlines and body text.
- Use nature-inspired icons for data inputs and visualizations.
- A clean, organized layout that emphasizes key prediction data with supporting visualizations.
